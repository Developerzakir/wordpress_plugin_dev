== Description ==
Wordpress Scroll to top plugin will help you to enable Back to Top button to your WordPress website.

= Usage =

* Go to your Dashboard after installation and navigate to "Appearance > Customize > Scroll To Top" to configure the accordion.

= Features =

  * Very easy installation
  * Flexible and easy to use


== Installation ==
1. Upload `wpbd-scroll-to-top-up` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
